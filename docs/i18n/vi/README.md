# Project Brain — tài liệu tiếng Việt

Thư mục này là lớp i18n tiếng Việt dành cho người dùng. Nguồn hướng dẫn canonical/internal nằm trong `.ai/` và được viết bằng tiếng Anh.

## Quy ước ngôn ngữ

- Input gửi vào agent: tiếng Anh.
- Hướng dẫn canonical cho agent: tiếng Anh.
- Output agent trả về người dùng: tiếng Việt.
- Lệnh, path, schema key và code identifier: giữ nguyên.
